# CONFIGURADOR DE KIT MOSQUERO - DOCUMENTO FINAL

## RESUMEN DE LÓGICA

El configurador guía al usuario a través de preguntas para determinar:
1. **Número de línea/caña** (#4, #5, #6 o #8)
2. **Nivel de equipo** (Entrada, Intermedio, Avanzado, Tope de gama)
3. **Tipo de línea** (Flotante o hundimiento, según técnica)

---

## 1. LÓGICA DE SELECCIÓN DE CAÑA

### 1.1 Salmón → Forzar #8
Si el usuario elige:
- "Salmones o salmonídeos de gran tamaño"
- **Resultado automático: #8**

### 1.2 Ríos pequeños + técnica fina → Forzar #4
Se recomienda caña #4 cuando:
- Caudal: pequeño
- Técnica: moscas secas o ninfas livianas
- Especie: peces pequeños a medianos
- Viento: bajo o moderado
- **Resultado: #4**

### 1.3 Decisión entre #5 y #6 por puntaje

#### Puntaje que suma hacia #6:
| Factor | Condición | Puntos |
|--------|-----------|--------|
| Caudal | Mediano | +1 |
| Caudal | Grande | +2 |
| Viento | Mucho viento | +2 |
| Especie | Peces medianos a grandes / streamers | +2 |
| Técnica | 50/50 ninfa + streamer | +1 |
| Técnica | Streamers | +2 |

#### Regla final:
- **Total ≥ 3 puntos → #6**
- **Total < 3 puntos → #5**

## 2. PRODUCTOS - CAÑAS (POR NIVEL)

### 2.1 Entrada
- **Fario** - Para aprendizaje, uso general, buen precio

### 2.2 Intermedio
- **ECHO Lift** - Más recuperación, precisión, distancia

### 2.3 Avanzado
- **ECHO Ion XL** - Acción más rápida, mejor para viento, streamers

### 2.4 Tope de gama
- **Líneas #3 a #6** → ECHO Boost Fresh (versátil, técnicas finas, foam)
- **Líneas #6 a #10** → ECHO Boost Blue (potencia máxima, viento, lago, salmón)

> En #6 se pueden ofrecer AMBAS:
> - Boost Fresh #6 → técnicas más finas
> - Boost Blue #6 → potencia y grandes moscas

#### Opción 1 - Presentación moderada
- **SA MPX #6** (flotante)
- Usar cuando: moscas tamaño medio, viento moderado/bajo, ninfas o secas grandes

#### Opción 2 - Potencia máxima
- **SA Titan #6** (flotante)
- Usar cuando: moscas grandes/voluminosas, foam (Chernobyl, Fat Albert, terrestres), viento fuerte

- **SA 25 Cold 300 grains**
- Cabeza corta, hundimiento rápido, tubos, streamers grandes, viento fuerte

## 6. FLUJO DEL CONFIGURADOR (PREGUNTAS)

### Paso 2: Tipo de agua / Caudal

### Paso 3: Técnica principal
- [ ] Moscas secas
- [ ] Ninfas

- [ ] Streamers principalmente
- [ ] Foam / Terrestres grandes

### Paso 4: Condiciones de viento

- [ ] Viento moderado
